# ☕ Coffee Shop Daily Orders

A Python-based coffee shop sales analysis application that demonstrates core Python programming concepts through a real-world business scenario. The program records daily coffee sales, calculates daily revenue using **lambda functions** and **built-in functions**, and demonstrates **variable shadowing** by comparing local and global variable scope.

This project was developed as part of the **University of the People – CS 1101: Programming Fundamentals** coursework while following software engineering practices such as modular code, documentation, and clear program output.

---

## 📌 Features

* ☕ Track daily coffee orders
* 💰 Calculate daily revenue
* 📈 Identify the busiest sales day
* 🔍 Demonstrate variable shadowing
* ⚡ Use lambda expressions with `map()`
* 🧰 Apply Python built-in functions such as `max()`
* 🖥 Beginner-friendly console interface
